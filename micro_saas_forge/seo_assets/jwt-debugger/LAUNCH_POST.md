Hey everyone, I just launched a small tool I built to scratch my own itch.

As a developer, I often found myself manually decoding and validating JWT tokens during auth debugging. It was tedious, error-prone, and a potential security risk if I missed something in the payload or signature.

So I built **JWT Debugger**. It’s a simple, free web tool that lets you paste a token, instantly decode its header and payload, and visually verify its structure. It helps spot common issues like incorrect algorithms or expired timestamps.

I’m hoping it saves other devs some time and headache. You can try it here:  
[https://jwt-debugger-l2c6kriko-tiankongmumus-projects.vercel.app](https://jwt-debugger-l2c6kriko-tiankongmumus-projects.vercel.app)

I’d really appreciate any feedback—especially if you run into bugs or have ideas for features. Thanks for checking it out.
