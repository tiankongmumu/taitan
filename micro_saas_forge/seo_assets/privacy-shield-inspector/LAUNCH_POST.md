Hey everyone, I just launched a side project I built to solve a problem I kept running into.

As a developer, I found it tedious to manually check if my sites had proper privacy and security headers (like CSP, HSTS, Permissions-Policy). Missing these can leave sites vulnerable and cause compliance headaches.

So I built **Privacy Shield Inspector**. It’s a simple, free tool that scans any public URL and gives you a plain-English breakdown of its security headers—what’s present, what’s missing, and how to fix gaps.

I’d love for you to try it and let me know what you think. Is it clear? Are there other headers or checks that would be useful?

You can test it here: https://privacy-shield-inspector-9s0rluugy-tiankongmumus-projects.vercel.app

Any feedback is greatly appreciated. Thanks!
