Hey everyone, I just launched a side project I built to scratch my own itch.

As a developer, I kept getting frustrated testing WebSocket APIs. Existing tools felt clunky for real-time debugging, and spinning up realistic mock servers was way too slow, which really stalled integrations.

So I built **WebSocket TestLab**. It’s a simple, no-frills web app that lets you connect to any WebSocket endpoint, send/receive messages in real-time, and quickly create mock servers with custom response logic—all in the browser.

I’m hoping it can save others some time. It’s completely free to use.

I’d genuinely appreciate any feedback from the community, especially on the mock server functionality. You can try it out here:
