Hey everyone, I just launched a small tool I built to scratch my own itch: the JSON Path Tester.

I often found myself tediously testing JSONPath expressions while working with APIs or data structures. Manually validating paths and checking outputs was slowing me down, and I wanted real-time feedback.

So I built a simple web app where you can paste any JSON, write your JSONPath expression, and see the matched results instantly. It’s meant to be fast, straightforward, and helpful for debugging or learning JSONPath.

You can try it here:  
https://json-path-tester-mqgeanr00-tiankongmumus-projects.vercel.app

I’d really appreciate any feedback—whether it’s about usability, features you’d find helpful, or anything I might have missed. Thanks for checking it out!
