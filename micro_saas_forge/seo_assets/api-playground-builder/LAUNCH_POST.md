Just launched a side project I built to scratch my own itch: API Playground Builder.

As a developer, I kept wasting time manually writing code just to test or prototype API endpoints. I wanted a quicker way to visualize responses, tweak parameters, and share configurations without spinning up a full project.

So I built a simple web tool where you can paste an OpenAPI spec (or start from a template) and instantly get a live, interactive API playground. You can test endpoints, see real responses, and share the playground with a link.

It’s early and rough around the edges, but I’d love any feedback from fellow devs.

Check it out here: https://api-playground-builder-hwpfawkzt-tiankongmumus-projects.vercel.app
