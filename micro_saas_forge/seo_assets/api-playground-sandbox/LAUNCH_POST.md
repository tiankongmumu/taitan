Just launched a small tool I built to scratch my own itch: API Playground Sandbox.

As a developer, I got tired of constantly setting up Postman collections or writing throwaway scripts just to test a simple REST endpoint. I wanted something faster—a clean, visual interface where I could paste a URL, tweak headers/body, and see the response instantly, without any setup.

So I built it. It’s a simple, no-login web sandbox for testing APIs. You can switch methods, edit JSON, add headers, and see formatted responses right away. It’s all client-side, so nothing is stored.

If you ever need to quickly check an endpoint or debug a response, give it a try. I’d really appreciate any feedback.

You can test it here: https://api-playground-sandbox-70loxf7af-tiankongmumus-projects.vercel.app
