Just launched a side project to solve a personal pain point: understanding complex code without sending it to a cloud AI.

It’s a **Privacy-First AI Code Explainer** that runs entirely in your browser. Your code never leaves your machine—it uses a local model via WebLLM. Paste a snippet, get a plain-English explanation.

I built this because I was uncomfortable pasting proprietary code into external services. It’s still early, but it works for many common languages and patterns.

Would love for you to try it and share any feedback. Does it explain your code clearly? What features would make it useful for your workflow?

Check it out here: https://privacy-first-ai-code-explainer-k3oyflaql.vercel.app
