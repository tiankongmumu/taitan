Hey everyone, I just built a small tool to solve a personal annoyance.

I often need to create temporary accounts for trials, public Wi-Fi logins, or one-off services. Using my main password feels risky, and I never remember the random ones I make up.

So I built **Privacypass Generator**. It creates secure, disposable passwords on the fly. You can set the length and character types, and it generates a strong passphrase locally in your browser—nothing is ever sent to a server.

It’s simple, free, and just a static page. I use it all the time now.

If you ever find yourself in a similar spot, give it a try:
