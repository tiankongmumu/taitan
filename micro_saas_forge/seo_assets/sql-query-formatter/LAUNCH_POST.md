Hey everyone, I just launched a small tool I built for myself and thought others might find it useful.

It’s a free SQL Query Formatter. I often work with messy, unformatted SQL from logs or legacy code, and it’s a pain to manually indent and structure it for readability. This tool quickly cleans it up, making debugging and collaboration much easier.

It’s a simple web app—just paste your SQL and get a formatted version. I built it to be fast and straightforward, with no sign-up required.

You can try it here: https://sql-query-formatter-6fh354oza-tiankongmumus-projects.vercel.app
