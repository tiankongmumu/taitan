Just launched a side project I’ve been working on: SQL Optimizer AI.

It’s a simple tool that helps improve slow or expensive SQL queries, especially for cloud databases like BigQuery, Snowflake, or Redshift. I kept running into issues where inefficient queries were driving up compute costs and hurting performance, so I built this to help spot common optimization opportunities—things like missing indexes, unnecessary joins, or heavy full-table scans.

You paste in your SQL, and it suggests rewritten versions along with a brief explanation of the changes. It’s completely free and runs in the browser (no sign-up).

I’d really appreciate any feedback from folks who work with SQL regularly—does it catch the right issues? Are the suggestions helpful?

You can try it here: https://sql-optimizer-nzdcl29x9-tiankongmumus-projects.vercel.app
