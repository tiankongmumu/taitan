Hey everyone, I just launched a small tool I built to solve a personal pain point.

Working with complex JSON APIs, I often found myself writing tedious loops or nested accesses just to extract a single deeply nested value. It was slow and easy to make mistakes.

So I built **JSON Path Finder**. You paste in your JSON, click on the value you want, and it instantly gives you the precise path (like `$.users[0].address.city`). You can also type a path to highlight the corresponding data.

It’s a simple, single-page web app—no login, no data stored. I use it almost daily now for debugging and exploration.

If you ever wrestle with nested JSON, I’d love for you to try it and share any feedback:
