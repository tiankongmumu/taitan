Hey everyone, I just launched a side project to solve a small but frequent pain point I had.

As a designer who often works with CSS, I found creating and previewing animated gradients to be a clunky process—juggling between code, color pickers, and the browser to get the timing and angles right. So I built the **CSS Gradient Animator**.

It’s a simple, real-time tool that lets you visually build smooth CSS gradient animations. You can adjust colors, stops, angles, duration, and easing on the fly, then copy the clean CSS code with one click.

I’d love for you to try it and let me know what you think—especially if there are features or controls that would make it more useful for your workflow.
