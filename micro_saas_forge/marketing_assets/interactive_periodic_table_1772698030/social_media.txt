=== REDDIT POST ===
**Title:** I was frustrated with static periodic tables, so I built an interactive one with real-time filtering and 3D models

Hey r/SideProject,

I was always frustrated with how static most online periodic tables felt when I was trying to study chemistry. It was hard to compare properties across elements quickly, and visualizing things was a pain. I wanted something that could show me trends instantly, not just individual data points.

So, I decided to build my own interactive periodic table. My main goal was to make it super easy to filter elements based on properties like density, electronegativity, or atomic radius in real-time. I wanted to see how elements group together based on these values, which I found incredibly helpful for understanding patterns. I also added a simple 3D visualization for each element, which I think helps give a better sense of their structure, and some quick quizzes for gamified learning – because who doesn't love a good quiz to test their knowledge?

Under the hood, it's a Next.js app, built with TypeScript. For the 3D models, I'm using React Three Fiber, which made integrating Three.js a lot smoother than I expected. All the filtering logic is client-side, processing a JSON dataset of element properties, so it feels pretty snappy.

You can check it out here: https://shipmicro.com/tools

This is still a work in progress, and I'd really appreciate any feedback, especially the brutal kind. What works, what doesn't, what's broken, what could be improved? Are there any other properties you'd find useful to filter by, or features you'd expect from a tool like this?

Thanks for taking a look!
