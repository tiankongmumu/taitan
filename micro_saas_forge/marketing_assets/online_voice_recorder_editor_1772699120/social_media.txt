=== REDDIT POST ===
**I was so frustrated with having to download bulky software just to record and trim a quick voice memo, so I built a browser-based voice recorder/editor.**

I often need to record short audio clips for work—feedback, quick notes, or snippets for presentations—and I got tired of installing (and updating) dedicated apps just to cut out a few seconds of silence or a stumble. I wanted something that worked right in the browser, required zero installs, and let me export a clean MP3 or WAV without jumping through hoops.

So I made a simple online tool that does exactly that:  
- Record directly in the browser using the MediaRecorder API  
- Basic trim/cut/splice functions (just enough to clean up a recording)  
- Export to MP3 (via libmp3lame.js) or WAV, no server processing—everything happens client-side  

The stack is pretty straightforward: Next.js for the frontend, TypeScript, and a bit of Web Audio API manipulation for the editing. The whole thing is static-hosted; no audio ever touches our servers, which felt like the right call for privacy and simplicity.

If you ever need to quickly record and trim voice audio without installing anything, you can try it here:  
[https://shipmicro.com/tools](https://shipmicro.com/tools)

I’d love some brutal feedback—especially if you run into any weird browser quirks or have ideas for small, useful features that wouldn’t overcomplicate it.
