## 🚀 极简上手 3 步走

### 步骤一：安装环境 (只需做一次)
1. 你的电脑需要安装 Python 3.10 以上版本。前往官网下载 `python.org`，安装时记得勾选 **"Add Python to PATH"**。
2. 打开电脑的命令行（按 `Win+R`，输入 `cmd` 回车）。
3. 复制并运行以下命令安装必备依赖库：
   ```cmd
   pip install playwright loguru
   playwright install chromium
   ```

### 步骤二：登录你的发文小号
本工具完全模拟真人操作，绝对安全。
1. 双击运行文件夹里的 `xhs_auto_publisher.py` (或者在命令行输入 `python xhs_auto_publisher.py`)。
2. **第一次运行非常重要！** 弹出的浏览器窗口会提示你需要登录小红书，请用**手机扫码登录**你的发文号。
3. 登录成功后，关闭黑窗口。系统已经记录了你的登录状态，以后就再也不用扫码了！

### 步骤三：开启全自动印钞客流机
以后每天出门上班前：
1. 双击运行本脚本。
2. 你什么都不用管，脚本会自动去后台抓取热门副业爽文，**自动加入防查重随机词、自动附带 AI 合规水印**。
3. 它会自己配好图并自动点击发布。完成后自动退出。你就等着看每天带来的私信流量吧！

> 💡 **防封高级技巧（必看）**：
> 千万不要再在文章里写“加微信 xxxx”，这年代小红书查得极严。
> 我们已经把脚本的话术改成了：**“想要保姆级教程的姐妹，主页置顶懂的都懂”**。
> 你只需要在小红书把置顶那一篇笔记加上你的微信号水印图，即可完美做到 0 风险引流。
